Quick Installation

1

Clone the Repository

git clone https://github.com/markfietje/jetson-kernel-modules-public.git

Download the kernel modules to your Jetson device

2

Navigate to Directory

cd

Move into the downloaded directory

3

Run Installation Script

sudo ./install.sh

The script will automatically install and configure everything

4

Verify Installation

lsmod | grep -E '(xfs|nfsd)'

Check that the modules are loaded successfully

Included Kernel Modules

XFS Filesystem

High-performance 64-bit journaling filesystem with support for large files and parallel I/O

32MB fs/xfs/xfs.ko

NFS Server

Network File System server for sharing files over the network with NFSv4 support

9.1MB fs/nfsd/nfsd.ko

Network Lock Manager

Manages file locking across networked file systems for data consistency

5.7MB fs/lockd/lockd.ko

RPC Support

Remote Procedure Call protocol support for network communication

10MB net/sunrpc/sunrpc.ko

GSS Authentication

Secure authentication for NFS using RPCSEC GSS

2.4MB net/sunrpc/auth_gss/auth_rpcgss.ko

Support Libraries

Additional modules for grace periods, ACL support, and CRC32c calculations

800KB lib + fs/nfs_common

Usage Guide

Mounting XFS Filesystems

Create Mount Point

sudo mkdir -p /media//

Mount the XFS Drive

sudo mount /dev/ /media//

Replace /dev/<device> with your actual device path and /<username>/<drive-label> with your preferred mount location

Verify Mount

mount | grep xfs

Configuring NFS Server

Create Export Directory

sudo mkdir -p /srv/nfs-share

Set Permissions

sudo chown nobody:nogroup /srv/nfs-share
sudo chmod 755 /srv/nfs-share

Add to Exports

echo "/srv/nfs-share *(rw,sync,no_subtree_check)" | sudo tee -a /etc/exports

Reload NFS Service

sudo exportfs -ra
sudo systemctl restart nfs-kernel-server

Troubleshooting

Modules not loading?

uname -r
sudo depmod -a
dmesg | grep -E '(xfs|nfsd)'

NFS server not starting?

sudo systemctl status nfs-kernel-server
sudo systemctl restart rpcbind

Mount errors?

sudo fdisk -l
sudo blkid /dev/sdXn
sudo mount -t xfs /dev/sdXn /mnt/test

Client can't connect?

showmount -e localhost
sudo ufw status
sudo exportfs -v
